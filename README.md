# Vue.js Todoapp

"Todo" App using Vue.js, Mongoose and Express.js

How to check it out,

Clone project then go to /server and
>npm install
>npm start
Then /client and
>npm install
>npm start dev
and navigate to
>localhost:8080/posts

Please note that you must have MongoDB installed to default port.
